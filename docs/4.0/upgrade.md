# Upgrade guide

We'll update this page when we release new Avo 4 versions.

If you're looking for the Avo 3 to Avo 4 upgrade guide, please visit [the dedicated page](./avo-3-avo-4-upgrade).

## Avo 4: `avo-pro` and `avo-advanced` split into feature gems

`avo-pro` and `avo-advanced` are being phased out. Add the individual feature gems you need instead. See [`avo-pro` and `avo-advanced` split into feature gems](./avo-3-avo-4-upgrade#avo-pro-and-avo-advanced-split-into-feature-gems) in the upgrade guide for the full checklist.
