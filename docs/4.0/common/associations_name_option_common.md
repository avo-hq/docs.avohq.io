<Option name="`name`">
Changes the text displayed as association name.

<Image src="/assets/img/4_0/common/associations/name-option.png" dark-src="/assets/img/4_0/common/associations/name-option-dark.png" width="2144" height="696" alt="A has_many association panel whose title uses a custom name option" />

#### Default

Plural association name.

#### Possible values

Any string or any zero arity lambda function.

Within lambda, you have access to all attributes of [`Avo::ExecutionContext`](../execution-context.html).
</Option>
