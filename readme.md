### Contributing

Docs for working on the Avo documentation site now live in the **Contributing** section of the docs themselves:

- [Running the docs site](https://docs.avohq.io/contributing/running-the-docs-site.html) — run it locally, available components, frontmatter helpers, and the field docs structure.
- [Writing Avo docs](https://docs.avohq.io/contributing/writing-docs.html) — how we structure a feature as a guide + reference (API) pair.

Source for those pages: [`docs/contributing/`](./docs/contributing/). Agents follow [`AGENTS.md`](./AGENTS.md).

Quick start:

```bash
yarn install && yarn dev
```

The site will be available on `http://localhost:3011/`.
